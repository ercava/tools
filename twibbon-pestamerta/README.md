# Twibbon Pesta Merta

This is a simple web application to generate custom twibbons for PESTAMERTA, the 2nd Ercaversary. users can upload their photo, adjust it within the frame, and download the result.


## How to use
1. Open the website.
2. Click on the frame to upload your photo.
3. Adjust the photo position and zoom.
4. Click "Unduh Twibbon" to download.
5. Copy the caption provided and post!
